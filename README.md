# **CALCULADORA DE IMC** 

<img width="800" src="https://github.com/GabrielaSozzo/calculadora-de-imc/blob/main/assets/images/calculadore-imc.PNG" />

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript. A aplicação permite que o usuário informe seu peso e altura para calcular automaticamente o IMC e exibir a classificação correspondente (como abaixo do peso, peso ideal, sobrepeso, obesidade, obesidade moderada, obesidade severa e obesidade morbida).

O projeto foi criado com o objetivo de praticar manipulação do DOM, validação de dados e aplicação de lógica condicional em JavaScript, além de aprimorar habilidades de estilização com CSS e construção de interfaces responsivas.
